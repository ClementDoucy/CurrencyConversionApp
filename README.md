# CurrencyConversionApp

Currency conversion and currency plotting created with Node and React.

To run back-end:
```
$ cd back
$ npm install
$ npm run build
$ npm run start
```
To run front-end:
```
$ cd front
$ npm install
$ npm run start
```

Or run using Docker:
```
$ docker-compose up
```

back-end endpoints:

`POST /api/convert`
`POST /api/history`

front-end features:

- Currency conversion

- history exchanges rates plotting on 1 year
